1. install nvm< saya pakai versi 18.20.4, tulis di cmd:
nvm install 18.20.4

2. buat folder proyek baru, tulis di cmd:
mkdir nodejs-axios-sample
cd nodejs-axios-sample

3. inisialisasi projek dengan npm 
init -y

4. Install Dependencies
npm install express axios typeorm sqlite3 body-parser
npm install --save-dev typescript ts-node @types/node @types/express @types/body-parser

5.  Inisialisasi TypeScript
npx tsc --init
